# taskmaster_als
Aplicación web de gestión de proyectos y tareas diseñada siguiendo el modelo de Aprendizaje Basado en Problemas (ALS). La aplicación permite a los usuarios organizar su flujo de trabajo mediante una jerarquía estructurada de categorías, proyectos y tareas, garantizando la persistencia y la integridad de los datos en una base de datos NoSQL.
